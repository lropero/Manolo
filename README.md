A poker (Texas hold'em) data generator for testing Reinforcement Learning algorithms.

### Requires

- [Node v18.14.1](https://nodejs.org/)
- npm v9.5.0

### Installation

```sh
$ npm ci
```

### Configuration

- Refer to [config/index.js](https://github.com/lropero/manolo/blob/master/config/index.js).

### Usage

```sh
$ npm run start
```
